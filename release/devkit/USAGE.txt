QUICK START — PLAYERS
---------------------

1. Copy EFL.dll into:
     <game folder>\mods\aurie\

2. Create this folder if it does not exist:
     <game folder>\mods\efl\

3. Drop any .efpack content packs into that folder.

4. Launch Fields of Mistria.

See Docs\EFL_Player_Guide.pdf for full instructions and troubleshooting.


QUICK START — MOD AUTHORS
--------------------------

1. Install EFL.dll as above.

2. Launch efl-devkit.exe.

3. On first run, set your Projects folder (where your pack folders live)
   and your Output folder (where .efpack files will be written).

4. Select a pack from the list, then use:
     Validate       — check manifest and content files for errors
     Pack → .efpack — build the distributable archive
     Watch + Repack — auto-rebuild on every file save (live development)

5. Copy the built .efpack into <game>\mods\efl\ to test it.

See Docs\EFL_DevKit_Guide.pdf for the full DevKit reference.
See Docs\EFL_Authoring_Guide.pdf to learn the content pack format.
See Docs\EFL_Example_Mod_Guide.pdf for a complete worked example.


REQUIREMENTS
------------

- Fields of Mistria (Steam or itch.io, version 1.0 or later)
- Aurie (native module loader) — install via MOMI
- MOMI (Mods of Mistria Installer)

EFL DevKit runs on Windows 10/11 with no additional dependencies.


DIAGNOSTIC CODES
----------------

If something fails to load, check the EFL log at:
  <game folder>\EFL\logs\efl.log

Codes follow the format  CATEGORY-SEVERITY###  (e.g. MANIFEST-E001).
A full code reference is in Docs\EFL_Authoring_Guide.pdf.
